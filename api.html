Tentang API Shikimori-MD

API Shikimori-MD memungkinkan Anda mengintegrasikan bot dengan sistem eksternal, aplikasi web, atau aplikasi mobile Anda. Dengan API ini, Anda dapat mengirim dan menerima pesan, mengelola bot, dan mengakses berbagai fitur Shikimori-MD secara terprogram.

RESTful API

Endpoint API modern dengan standar RESTful untuk integrasi mudah

Keamanan

Autentikasi JWT dan rate limiting untuk perlindungan sistem

Dokumentasi Lengkap

Dokumentasi API detail dengan contoh untuk semua bahasa populer

Autentikasi API

Untuk menggunakan API Shikimori-MD, Anda memerlukan API Key yang bisa didapatkan melalui:

// Dapatkan API Key melalui bot WhatsApp
/apikey generate

// Atau melalui dashboard web
https://dashboard.shikimori-md.com/api

Gunakan API Key di header semua request:

Authorization: Bearer YOUR_API_KEY

Catatan: Jaga kerahasiaan API Key Anda. Jika API Key terkompromi, segera generate yang baru dan nonaktifkan yang lama.

Endpoint Utama

Berikut beberapa endpoint utama yang tersedia:

1. Mengirim Pesan

POST /api/v1/messages/send
{
  "to": "6281234567890",
  "message": "Hello from API!",
  "type": "text" // text, image, video, document, etc.
}

2. Menerima Pesan (Webhook)

POST [YOUR_WEBHOOK_URL]
{
  "event": "message",
  "from": "6281234567890",
  "message": "Pesan dari user",
  "timestamp": 1634567890
}

3. Manajemen Grup

POST /api/v1/groups/create
{
  "name": "Nama Grup",
  "participants": ["6281234567890", "6289876543210"]
}

Contoh Implementasi

Berikut contoh kode untuk mengirim pesan menggunakan API Shikimori-MD:

JavaScript (Node.js)

const axios = require('axios');

async function sendMessage() {
  try {
    const response = await axios.post(
      'https://api.shikimori-md.com/api/v1/messages/send',
      {
        to: '6281234567890',
        message: 'Hello from Node.js!',
        type: 'text'
      },
      {
        headers: {
          'Authorization': 'Bearer YOUR_API_KEY'
        }
      }
    );
    console.log(response.data);
  } catch (error) {
    console.error(error);
  }
}

sendMessage();

Python

import requests

url = "https://api.shikimori-md.com/api/v1/messages/send"
headers = {
  "Authorization": "Bearer YOUR_API_KEY"
}
data = {
  "to": "6281234567890",
  "message": "Hello from Python!",
  "type": "text"
}

response = requests.post(url, json=data, headers=headers)
print(response.json())

Rate Limiting

Untuk menjaga stabilitas sistem, API Shikimori-MD memiliki batasan request:

  • Free Tier: 60 request per menit
  • Pro Tier: 300 request per menit
  • Enterprise: Custom limit berdasarkan kebutuhan

Response headers akan selalu menyertakan informasi rate limit Anda:

X-RateLimit-Limit: 60
X-RateLimit-Remaining: 59
X-RateLimit-Reset: 1634567890
