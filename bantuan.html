Pertanyaan Umum

Bagaimana cara mengundang bot ke grup WhatsApp saya?

Untuk mengundang Shikimori-MD ke grup WhatsApp Anda:

  1. Buka grup WhatsApp yang ingin Anda tambahkan bot
  2. Klik "Tambah Anggota" di menu grup
  3. Cari nomor bot (+62xxx) atau scan QR code dari website resmi
  4. Pastikan Anda memiliki izin admin untuk menambahkan anggota baru

Setelah ditambahkan, bot akan mengirimkan pesan pengenalan dan daftar perintah dasar yang tersedia.

Mengapa beberapa perintah tidak bekerja di grup saya?

Beberapa kemungkinan penyebab dan solusinya:

  • Izin Admin: Beberapa perintah membutuhkan status admin. Pastikan bot telah dijadikan admin grup.
  • Pembatasan Grup: Fitur mungkin dinonaktifkan oleh pengaturan grup. Gunakan perintah /settings untuk mengecek.
  • Pembaruan Bot: Fitur mungkin sedang dalam perbaikan. Cek status di /status.
  • Limit Penggunaan: Akun gratis memiliki batasan perintah tertentu.
Apakah data percakapan saya disimpan oleh bot?

Kebijakan penyimpanan data Shikimori-MD:

  • Data yang Disimpan: ID pengguna, preferensi setting, log aktivitas terbatas (waktu penggunaan, perintah yang digunakan)
  • Data Tidak Disimpan: Konten percakapan biasa, media yang dikirim (kecuali untuk fitur tertentu yang membutuhkannya)
  • Penyimpanan Sementara: Beberapa fitur seperti reminder atau notes menyimpan data secara sementara hingga fitur selesai dijalankan

Anda dapat meminta penghapusan data melalui perintah /deleteaccount atau menghubungi tim dukungan.

Bagaimana cara melaporkan bug atau memberikan saran?

Kami menghargai semua laporan dan saran Anda:

  1. Grup Komunitas: Bergabunglah dengan grup WhatsApp resmi kami untuk diskusi langsung
  2. Formulir Online: Isi formulir laporan bug di website resmi Shikimori-MD
  3. Media Sosial: Hubungi kami melalui Twitter @ShikimoriMD_Support

Sertakan detail seperti: versi bot, langkah reproduksi bug, screenshot, dan perangkat yang digunakan.
