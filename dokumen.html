Shikimori MD

A powerful JavaScript wrapper for the Shikimori API

Introduction

Shikimori MD is a modern JavaScript library that provides easy access to the Shikimori API, allowing developers to fetch anime, manga, and user-related data with minimal setup.

Note: This library requires Node.js 14+ or modern browser with ES6 support.

Installation

Using yarn

yarn add shikimori-md

Using npm

npm install shikimori-md

CDN (Browser)

<script src="https://cdn.jsdelivr.net/npm/shikimori-md@latest/dist/shikimori-md.min.js"></script>

Quick Start

// ES6 Import
import Shikimori from 'shikimori-md';

// CommonJS
const Shikimori = require('shikimori-md');

// Initialize
const shiki = new Shikimori();

// Get anime by ID
shiki.anime.getById(1)
    .then(anime => console.log(anime))
    .catch(err => console.error(err));

API Reference

Anime

GET /anime/:id

Get anime by ID

shiki.anime.getById(id);
GET /anime/search

Search anime by query

shiki.anime.search({ q: 'Naruto', limit: 5 });

Manga

GET /manga/:id

Get manga by ID

shiki.manga.getById(id);

Users

GET /users/:id

Get user by ID or nickname

shiki.users.getById('username');

Authentication

For authorized endpoints, you need to initialize the client with your OAuth credentials:

const shiki = new Shikimori({
    clientId: 'YOUR_CLIENT_ID',
    clientSecret: 'YOUR_CLIENT_SECRET',
    redirectUri: 'YOUR_REDIRECT_URI'
});
