Dengan menggunakan bot Shikimori-MD ("Bot"), Anda setuju untuk terikat oleh Syarat Layanan ini. Harap baca dengan seksama sebelum menggunakan layanan kami.

CATATAN PENTING: Penggunaan Bot ini sepenuhnya tanggung jawab pengguna. Kami tidak bertanggung jawab atas penyalahgunaan atau pelanggaran yang dilakukan pengguna.

1. Persyaratan Penggunaan

1.1 Ketentuan Umum

  • Bot ini ditujukan untuk pengguna berusia minimal 13 tahun
  • Pengguna bertanggung jawab penuh atas aktivitas yang dilakukan melalui Bot
  • Dilarang menggunakan Bot untuk aktivitas ilegal atau melanggar hukum

1.2 Pembatasan Penggunaan

Anda setuju untuk TIDAK:

  1. Menggunakan Bot untuk spam atau flood pesan
  2. Memanipulasi atau mencoba meretas sistem Bot
  3. Menyebarkan konten ilegal, pornografi, atau kekerasan
  4. Melakukan aktivitas yang dapat membebani server
  5. Menggunakan Bot untuk penipuan atau phishing

2. Privasi dan Keamanan

2.1 Data Pengguna

Kami dapat menyimpan data berikut untuk keperluan operasional Bot:

  • Nomor telepon pengguna
  • Pesan dan perintah yang dikirim ke Bot
  • Data penggunaan (log aktivitas)

2.2 Perlindungan Data

Kami berkomitmen melindungi data pengguna dengan:

  1. Tidak menjual atau membagikan data ke pihak ketiga
  2. Menggunakan enkripsi untuk data sensitif
  3. Menyimpan data hanya selama diperlukan

3. Pembatasan Tanggung Jawab

Pengembang Bot tidak bertanggung jawab atas:

  • Kerugian atau kerusakan akibat penggunaan Bot
  • Konten yang dihasilkan oleh fitur pihak ketiga
  • Downtime atau gangguan layanan di luar kendali kami
  • Penyalahgunaan Bot oleh pengguna lain

4. Perubahan Syarat Layanan

Kami dapat mengubah Syarat Layanan ini kapan saja. Perubahan akan diberitahukan melalui pembaruan di situs resmi atau melalui Bot. Penggunaan berkelanjutan setelah perubahan berarti Anda menerima syarat baru.
