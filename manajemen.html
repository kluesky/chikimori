Manajemen Grup

Fitur Canggih untuk Mengelola Grup WhatsApp dengan Shikimori-MD

Fitur Manajemen Grup

Shikimori-MD menyediakan berbagai alat canggih untuk membantu Anda mengelola grup WhatsApp dengan lebih efektif dan efisien.

Moderasi Otomatis

Filter spam, kata kasar, dan konten tidak pantas secara otomatis

Asisten Grup

Bot akan membantu menjawab pertanyaan umum di grup

Analitik Grup

Statistik aktivitas anggota dan popularitas konten

Perintah Dasar

Berikut perintah-perintah dasar untuk manajemen grup:

1. Setup Bot di Grup

/group activate - Mengaktifkan bot di grup ini
/group deactivate - Menonaktifkan bot di grup ini
/group settings - Melihat pengaturan grup

2. Manajemen Anggota

/group add 6281234567890 - Menambahkan anggota
/group kick @mention - Mengeluarkan anggota
/group promote @mention - Memberi admin
/group demote @mention - Mencabut status admin

3. Fitur Grup

/group welcome on - Aktifkan pesan selamat datang
/group goodbye on - Aktifkan pesan selamat tinggal
/group rules - Atur peraturan grup
/group info - Informasi grup

Catatan: Hanya admin grup yang dapat menggunakan perintah-perintah di atas. Pastikan bot telah dijadikan admin grup untuk fitur lengkap.

Fitur Lanjutan

Fitur-fitur canggih untuk manajemen grup profesional:

  • Auto Responder: Jawab pertanyaan umum otomatis
  • Anti-Link: Blokir link dari domain tertentu
  • Word Filter: Sensor kata-kata tertentu
  • Backup Grup: Simpan data anggota dan chat penting
  • Jadwal Pesan: Kirim pengumuman sesuai jadwal
  • Anti-Spam: Deteksi dan blokir spam otomatis
  • Voting System: Buat polling langsung di grup
  • Auto-Translate: Terjemahkan pesan antar bahasa

Level Permission

Shikimori-MD memiliki sistem permission untuk mengontrol akses fitur:

Level Deskripsi Akses
Pemilik Pembuat grup Semua perintah
Admin Admin grup Hampir semua perintah kecuali beberapa fitur khusus
Anggota Member biasa Perintah dasar seperti info, help, dll
Terbatas Anggota yang dibatasi Tidak bisa menggunakan perintah bot
